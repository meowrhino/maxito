# Registro de Cambios v2

Este documento detalla las mejoras y refactorizaciones implementadas en la segunda versión del proyecto "maxito".

## 1. Refactorización Estética (Estilo Windows XP "Roto")

- **Paleta de colores:** Se ha implementado una nueva paleta de colores inspirada en Windows XP pero con tonos "rotos" para un look más personal:
    - Fondo principal: Blanco roto (`#f5f5f0`)
    - Barra lateral: Gris claro (`#d4d0c8`)
    - Bordes: Gris medio (`#808080`)
    - Proyecto activo: Verde suave (`#0a0`)
    - Flechas y links (hover): Azul suave (`#6b9bd1`)
- **Tipografía:** Se ha cambiado la fuente a **Helvetica** en toda la interfaz para una apariencia más limpia y moderna que la anterior monoespaciada.
- **Eliminación de bordes:** Se han eliminado los bordes de colores para un diseño más plano y minimalista.

## 2. Rediseño de Controles de Navegación

- **Eliminación de `slide-controls`:** Se ha eliminado la barra de controles central.
- **Nuevos controles fijos:**
    - **Flechas de navegación:** Ahora son más grandes y están fijas en las esquinas inferiores de la pantalla (`bottom: 2rem`, `left: 2rem` / `right: 2rem`).
    - **Contador de diapositivas:** Se ha movido a la esquina superior derecha, con un formato de fracción diagonal para un look más distintivo.
- **Indicadores de proyecto:** El texto que indica el proyecto anterior/siguiente ahora aparece junto a las flechas de navegación, mejorando la visibilidad y la usabilidad.

## 3. Mejoras en el Diseño Responsivo (Móvil)

- **Barra de navegación (`#sidebar`):**
    - Se ha limitado su altura máxima a `25dvh` para dejar más espacio al contenido principal.
    - Los botones de idioma y "maxito" (About) ahora están fijos en la esquina superior e inferior derecha respectivamente, sin afectar el layout del scroll.
    - La lista de proyectos (`#project-nav`) ahora es la única zona con scroll vertical, optimizando el espacio.
- **Contenido principal (`#slide-container`):**
    - Se han ajustado los `paddings` y `margins` para "apretar" el contenido y reducir la necesidad de scroll vertical en la medida de lo posible.
    - Las imágenes tienen una altura máxima de `35dvh` para asegurar que los controles y el texto sean siempre visibles.

## 4. Modal "About" como Popup

- **Diseño de popup:** El panel "About" ya no se desliza desde un lateral, sino que aparece como un modal centrado que no ocupa toda la pantalla.
- **Estilo:**
    - Tiene un borde de `2px` y una sombra (`box-shadow`) para darle profundidad y destacarlo sobre el contenido.
    - Se muestra sobre un velo blanco semi-transparente (`rgba(255, 255, 255, 0.8)`) que cubre el resto de la página.

## 5. Refactorización del Código

- **JavaScript (`app.js`):**
    - Se ha refactorizado la lógica para manejar los nuevos controles de navegación y el contador de diapositivas.
    - Se ha mejorado la función `updateProjectIndicators` para posicionar correctamente los textos junto a las nuevas flechas.
    - Se ha mantenido la estructura atómica y legible del código.
- **CSS (`style.css`):**
    - Se ha reescrito gran parte del CSS para implementar la nueva paleta de colores, la tipografía y el layout de los nuevos controles.
    - Se ha optimizado el CSS para el nuevo diseño responsivo, usando `dvh` para la barra lateral y ajustando los elementos fijos.
- **HTML (`index.html`):**
    - Se ha actualizado la estructura para reflejar los nuevos controles de navegación y se ha cambiado el nombre del botón "About" a "maxito".
    - Se ha mantenido el código limpio y semántico.
