# Hoja de Ruta del Proyecto: Maxito

Este documento describe la hoja de ruta para el desarrollo del sitio web "Maxito", un portafolio personal que fusiona conceptos de navegación y diseño de diferentes fuentes para crear una experiencia de usuario única.

## Fase 1: Análisis y Planificación (Completada)

- **Objetivo:** Entender los requisitos del proyecto, analizar los sitios web de referencia y definir la estructura del proyecto.
- **Tareas completadas:**
    - Análisis de la navegación de [mirandaperez.info](https://mirandaperez.info).
    - Análisis de la lógica de slides de [zumino.ppt](https://zumino.vercel.app/).
    - Análisis de la estética minimalista de [barbarawong.info](https://barbarawong.info/).
    - Definición de la estructura de datos para `data.json`.
    - Creación de la estructura de carpetas del proyecto.

## Fase 2: Generación de Contenido Visual (Completada)

- **Objetivo:** Crear las imágenes necesarias para los proyectos de ejemplo.
- **Tareas completadas:**
    - Generación de 13 imágenes de árboles con un estilo minimalista y variado para poblar las slides de los proyectos.

## Fase 3: Desarrollo del Prototipo Funcional (Completada)

- **Objetivo:** Desarrollar el sitio web con HTML, CSS y JavaScript, implementando las funcionalidades clave.
- **Tareas completadas:**
    - Creación del archivo `index.html` con la estructura principal.
    - Creación del archivo `style.css` con el diseño minimalista y responsivo.
    - Creación del archivo `app.js` con la lógica de la aplicación:
        - Carga y parseo de `data.json`.
        - Navegación entre proyectos y slides.
        - Implementación de la funcionalidad `?slug=`.
        - Gestión del panel "About".
        - Funcionalidad de cambio de idioma (preparada).
    - Creación del archivo `data.json` con los 3 proyectos de ejemplo.

## Fase 4: Documentación y Despliegue (Completada)

- **Objetivo:** Crear la documentación del proyecto y subir el código a un repositorio de GitHub.
- **Tareas completadas:**
    - Creación de este documento `ROADMAP.md`.
    - Creación del documento `TODO.md`.
    - Creación del repositorio "maxito" en GitHub.
    - Subida de todos los archivos del proyecto al repositorio.

## Próximos Pasos

- Revisión y feedback del cliente.
- Implementación de mejoras y ajustes basados en el feedback.
- Optimización de imágenes y código para producción.
- Configuración del despliegue final en un servicio de hosting (ej. GitHub Pages, Vercel, Netlify).
